			$f(x_1) = f(x_2) = 0$, so we have $f(x_1)\leq f(x_2)$ as wanted.

				1-\frac{1}{x_1} &< 1-\frac{1}{x_2}\\
				f(x_1) &< f(x_2)
			\end{aligned}$$
			As wanted.
